# AI and Machine Learning

This repository provides an introduction to Artificial Intelligence (AI) and Machine Learning (ML). It covers how ML differs from traditional rule-based programming, the main types of ML (supervised, unsupervised, and reinforcement learning), and the supervised learning workflow — ending with a hands-on exercise that trains a classification model on a tabular dataset.

## Goals

- Explain how machine learning differs from traditional rule-based programming.
- Differentiate between supervised, unsupervised, and reinforcement learning.
- Implement a simple supervised learning model on a tabular dataset.

The coding exercises are meant to illustrate these ideas. You don't need to memorize the syntax or be able to write any of this code from scratch — being able to read, modify, and reason about it (with help from tools like Claude or ChatGPT) is the goal.

## Lesson Materials

For the practical exercise, upload the `.ipynb` notebook to Google Colab to run it.

- `Lesson 01 - AI and Machine Learning.pptx` — Slide deck for the lesson.
- `Supervised_Learning.ipynb` — Practical exercise: train and compare basic supervised learning classifiers.
