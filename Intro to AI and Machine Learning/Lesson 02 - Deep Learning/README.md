# Deep Learning

This repository builds on the machine learning foundations from the previous lesson and introduces deep learning and modern AI. It covers how a single artificial neuron relates to logistic regression, how neurons combine into neural networks, and how different architectures suit different problems — then extends into modern topics including fine-tuning pretrained models and agentic AI, with hands-on exercises throughout.

## Goals

- Describe how an artificial neuron extends logistic regression.
- Explain how neural networks work.
- Identify different deep learning architectures for domain-specific problems.

The coding exercises are meant to illustrate these ideas. You don't need to memorize the syntax or be able to write any of this code from scratch — being able to read, modify, and reason about it (with help from tools like Claude or ChatGPT) is the goal.

## Lesson Materials

For the practical exercises, upload the `.ipynb` notebooks to Google Colab to run them.

- `Lesson 02 - Deep Learning.pptx` — Slide deck for the lesson.
- `Neural_Networks.ipynb` — Practical exercise: build an MLP and a CNN in PyTorch for a computer vision task.
- `Fine_Tuning.ipynb` — Practical exercise: experiment with Hugging Face and fine-tune a model for a language task.
- `Agentic_AI.md` — Walkthrough: build a data analyst agent in Azure AI Foundry.
- `heart.csv` — Pre-cleaned UCI Cleveland heart disease dataset used in the agent walkthrough.
